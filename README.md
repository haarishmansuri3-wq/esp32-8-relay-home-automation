# ESP32 8-Relay Home Automation

An ESP32-based IoT home automation system capable of controlling 8 electrical appliances through relay modules over Wi-Fi.

## Features

* Controls up to 8 relays using ESP32
* Wi-Fi-based home automation
* Can switch lights, fans, and appliances
* Expandable for mobile app or web control
* Designed for low-cost IoT kits and training

## Hardware Used

* ESP32
* 8-Channel Relay Module
* Power Supply
* Jumper Wires

## Repository Structure

* `gerber/` → PCB manufacturing files
* `firmware/` → ESP32 Arduino code
* `docs/` → PCB images and wiring diagrams
